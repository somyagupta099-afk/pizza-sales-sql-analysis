Pizza Sales Analysis using SQL
📌 Project Overview
This project analyzes pizza sales data using SQL to identify sales trends, customer preferences, and revenue insights.

The dataset contains order information, pizza details, and category data, which are analyzed using SQL queries including joins, aggregations, and window functions.

This project demonstrates how SQL can be used for real-world business analysis.

📂 Dataset Description
The dataset consists of four tables:

1️⃣ Orders
Contains information about order date and time.

2️⃣ Order_Details
Stores pizza ID and quantity for each order.

3️⃣ Pizzas
Contains pizza size and price information.

4️⃣ Pizza_Types
Contains pizza name, category, and ingredients.
